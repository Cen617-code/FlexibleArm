# FlexibleArm (多级柔性臂动力学仿真系统)

## 项目简介

本项目是一个基于 MATLAB 的多级柔性臂动力学仿真与可视化系统。系统基于拉格朗日方程建立了多级串联柔性臂的严格动力学模型，支持 1~N 级任意级数的串联仿真，并提供了高自由度的参数化配置接口。通过该模块，能方便地进行柔性机械臂的受力分析、运动轨迹规划与控制算法的验证。

## 目录结构

- `代码文件/`: MATLAB 仿真核心代码，包括主调度入口、动力学方程求解、几何变换计算等。
- `python_ui/`: 基于 Python 开发的图形化用户界面，用于快捷配置仿真参数并调用 MATLAB 仿真后端。
- `推导文档/` & `推导.pdf`: 包含柔性臂动力学模型的详细数学理论与推导过程。
- `结果图示/`: 仿真运行后输出的代表性结果图表留存。
- `说明文档.md`: 核心代码架构、调用关系及参数配置的详细说明文档。

## 快速开始

### 运行环境

- MATLAB (包含核心数学与微分方程求解模块，如 `ode45`)
- (可选) Python 环境及 MATLAB Engine for Python 依赖（如需使用 Python UI）

### 运行仿真

直接在 MATLAB 中运行入口脚本，即可体验默认配置（默认 8 级串联）下的多级柔性臂仿真：

```matlab
% 进入代码目录并运行
cd 代码文件
main
```

若需定制仿真参数，建议直接调用 `simulate_arm` 接口。例如，运行一个 4 级柔性臂的仿真，不输出图像，并附加自定义外力：

```matlab
result = simulate_arm('stageCount', 4, ...
                     'enablePlots', false, ...
                     'customForceMatrix', [0.5, 0, 0; NaN, NaN, NaN; 0.15, 0.08, 0.02; 0.1, 0, 0]);
```

## 核心模块与架构

系统主要划分为以下几个核心功能区域（以图示关系见及深入细节请阅读 `说明文档.md`）：

1. **驱动入口 (`main.m`)**: 极简启动脚本，提供零配置的“一键运行”体验。
2. **仿真编排 (`simulate_arm.m`)**: 新的顶层调度器。统一管理多级柔性臂的参数装配（`parseSimulationParameters`）、配置生成（`buildSimulationConfig`）、时间轴规划及可视化流程。
3. **动力学核心 (`arm_new.m`)**: 动力学演化的核心模块。摒弃了针对单级的硬编码假设，转为完全参数化的实现：按当前配置自适应装配质量矩阵、施加控制力、以及解算级联约束。
4. **几何与派生计算 (`chainTransforms.m`, `quan.m` 等)**: 提供多级受约束体系的统一姿态链描述。用于为主演化核心及后期图像呈现提供几何位姿支撑。

## 高级特性

- **动态级数伸缩**：只需修改 `stageCount`，系统自动复制模板、设置级间偏角（如默认 60° 偏转），自动扩展求解维度，并计算对应的广义坐标。
- **自定义控制框架**：外力设计与控制模块支持丰富的输入格式（常量向量、函数句柄、或者字符串表达式），可针对每一级单独下达覆写指令。
- **稳健的数值处理**：内置约束漂移修正与雅可比有限差分，根据系统条件数自适应选择最优的线性求解策略（`\` 或伪逆）。
- **良好的多语言扩展性**：核心封装于返回无状态 `result` 结构体的设计模式，极为便于和 Python 等外部程序交互协同。
